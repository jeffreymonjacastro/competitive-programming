## 📖 Description
Tipic problem of Binary Search but if a target number is not found, return the index where the target should be

## 💡 Approach
+ Make a simple binary search: `l = 0, r = size -1`
+ Then, look for the middle index
+ If the element is not found, then `r > l`, so the answer must be `l`

## ⚡ Complexity
- **Time:** O(logn)
- **Space:** O(1)
